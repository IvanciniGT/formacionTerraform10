

Nuestro despliegue:
- Crearemos una máquina virtual en AWS
- Para crear esa máquina usaremos una imagen de SO.. Ubuntu
- Pincharemos esa máquina en una subred, que crearemos dentro de una VPC
- VPC: Virtual Private Cloud (es un entorno de red virtual aislado dentro de AWS)
  En ella podemos crear subredes, tablas de ruteo, gateways, etc 

- Vamos a usar el proveedor oficial de AWS para Terraform
